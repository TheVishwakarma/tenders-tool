🚀 One-Click Daily Setup — Free GitHub Pages Hosting

  1. Go to github.com → sign up free → click New repository → name it tenders-tool → set to Public → click Create
  2. Click Add file → Upload files → drag this HTML file in → rename it index.html → click Commit changes
  3. Go to Settings → Pages → under Source select Deploy from branch: main / root → click Save
  4. Your tool is now live at https://YOUR-USERNAME.github.io/tenders-tool — bookmark this URL
  5. Each day: open the URL → upload bulletin PDF → upload Excel → generate → done. No install, no code, works on any browser
  6. If filter rules change: download updated HTML from this chat → go to your repo → click index.html → pencil icon → paste → Commit
